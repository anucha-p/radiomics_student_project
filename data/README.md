# Data Directory

Place your datasets here. The project supports several data options:

## Quick Start

### Option 1: Use Pre-extracted Features (Recommended for beginners)
```python
# In your notebook:
import pandas as pd
url = "https://raw.githubusercontent.com/aydindemircioglu/radMLBench/main/datasets/LNDb.csv.gz"
df = pd.read_csv(url, compression='gzip')
```

### Option 2: Generate Synthetic Data
```bash
cd scripts/
python sample_radiomics_dataset.py
```

### Option 3: Download Real radMLBench Datasets
```bash
cd scripts/
python download_radmlbench_datasets.py --list        # See available datasets
python download_radmlbench_datasets.py --dataset LNDb # Download specific dataset
```

## Expected Data Formats

### For Notebook 01 (Feature Extraction)
```
data/
└── synthetic/
    ├── Patient_001/
    │   ├── image.nii.gz    # 3D medical image
    │   └── mask.nii.gz     # Segmentation mask
    ├── Patient_002/
    │   └── ...
    └── labels.csv          # Patient labels
```

### For Notebook 02 (Machine Learning)
CSV file with columns: `ID`, `Target` (0 or 1), and feature columns (e.g., `original_firstorder_Mean`)
